# Voice samples

Drop Akila's past LinkedIn posts here (or other writing that sounds right) so future drafts can match the cadence.

## What to add

- Past LinkedIn posts that performed well or that Akila feels read like her — paste as `.md` files, one post per file
- Voice memos / transcripts where Akila talks about a topic the way she'd want to write about it
- Short do/don't lists ("words I'd never use", "phrases I overuse and want to cut")

## File naming

`YYYY-MM-DD_short-slug.md` for past posts. `do-dont.md` and `phrases-to-avoid.md` for rule files.

## Template for a sample

```
---
date: 2026-04-12
where: LinkedIn
performed: high | mid | low
note: what made this feel like Akila
---

[paste the post text here, exactly as published]
```

## Why this matters

The drafting agent reads this folder before writing. Three good samples beats a tone-of-voice paragraph.
