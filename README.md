# PurchasePlus Presentation Agent

AI-powered document-to-presentation converter. Converts PPTX, PDF, DOCX, and Google Slides into branded PurchasePlus HTML presentations.

## Usage

Open `index.html` (the intake page) to get started. Sign in with your @marketboomer.com Google account.

## Pages

- `index.html` / `intake.html` — upload & analyse documents (MasterChef agent)
- `editor.html` — edit and configure generated presentations
- `master-library.html` — master slide library
- `template/` — design system and base presentation template
